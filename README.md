# travelgram

**travelgram** é um projeto front-end desenvolvido utilizando HTML e CSS, com foco na prática de estilização e construção de layout em versão desktop.

## Índice

- [Visão Geral](#visão-geral)
- [Tecnologias Utilizadas](#tecnologias-utilizadas)

## Visão Geral

O **travelgram** foi desenvolvido com o objetivo de praticar CSS na construção de layouts organizados e visualmente atrativos. O projeto consiste em uma página de perfil de viagens, exibindo informações do usuário e uma galeria de fotos organizada em grid.

A aplicação foi construída em versão desktop, utilizando HTML para estruturar o conteúdo da página e CSS para estilização, alinhamento dos elementos e organização da galeria de imagens.

## Tecnologias Utilizadas

- **HTML**: Linguagem de marcação utilizada para estruturar o conteúdo da página.
- **CSS**: Linguagem de estilização utilizada para definir o layout e o design da aplicação.
